# 🇾🇪 Yemen Products Export

تصدير منتجات اليمن من OpenFoodFacts إلى ملفات Excel و CSV

## الملفات المتاحة

- `yemen_products_*.xlsx` - ملف Excel
- `yemen_products_*.csv` - ملف CSV

## المعلومات

- تحديث تلقائي كل أسبوع
- البيانات من OpenFoodFacts
- شامل الباركود والعلامات التجارية والفئات

## التعليمات

1. اذهب إلى Actions
2. اختر "Export Yemen Products"
3. اضغط Run workflow
4. حمل الملفات من Artifacts
